# ELO changes only for the live racer, not the Ghost Run creator

When a player races a Ghost Run, only the live racer's ELO changes. The Ghost Run creator's ELO is never affected by someone else racing their ghost. We chose this because a player should not lose rank from matches they are unaware of and cannot influence — having your rating tank overnight because many players raced your ghost would feel arbitrary and undermine trust in the ranking system. It also removes the incentive to deliberately record a slow ghost in order to drain opponents.

The Ghost Run carries a Stamped ELO (the creator's rating immediately after the run was recorded) which is used solely to calibrate how much ELO the racer gains or loses, not as a live stake in the outcome.
